# DOHA STORE

Web e-commerce premium para marca de ropa urbana.

## Tecnologías

* HTML
* CSS
* JavaScript

## Funcionalidades

* Catálogo de productos
* Carrito
* Compra por WhatsApp
* Diseño responsive
* Estética premium streetwear

## Deploy

Proyecto preparado para GitHub Pages.

## Instrucciones para subirlo a GitHub Pages

1. Crear repositorio en GitHub llamado `doha-store`.
2. Subir todos los archivos del proyecto.
3. Ir a `Settings > Pages`.
4. En `Branch` elegir `main`.
5. En carpeta elegir `root`.
6. Guardar.
7. Abrir el link generado por GitHub Pages.
